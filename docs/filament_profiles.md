# Filament profile schema

```yaml
filaments:
  - name: White PLA
    color_hex: "#F2F2F2"
    td_mm: 1.05
    notes: "semi translucent"
```

Fields:
- `name`: display name.
- `color_hex`: hex RGB.
- `td_mm`: transmission distance in mm.
- `notes`: optional operator comments.

Profiles may be YAML or JSON.
